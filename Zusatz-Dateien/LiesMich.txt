Dies ist die Version 2.0 von CV-Navi, der Programmiersoftware fuer die neueren Tams-Decoder (ab LD-30er) und andere Geraete.
Um das Programm zu starten reicht es, KlarText.jar zu starten (Doppel-Klick mit der Maus).
Bei Linux muss das Startscript CV-Navi.sh gestartet werden.
Wenn beim Setup angegeben, kann das Programm auch aus dem Start-Menue oder der Schnellstartleiste gestartet werden (Windows).
Das Programm ist eine Java-Anwendung. Es muss also zwingend Java auf dem PC installiert sein.
Bei Linux muss noch die rxtx-Bibliothek selbst installiert werden.
Sollte das nicht der Fall sein, kann das so genannte Java Runtime Environment (JRE) von der Webseite der Firma Sun/Oracle heruntergeladen und installiert werden.
